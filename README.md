Ozon Documentation
=======

## Web Access
https://docs-ozon.btech.id/


## Installation

### 1. Install python-pip
```
## Python2
wget https://bootstrap.pypa.io/get-pip.py
python get-pip.py

## Python 3
sudo apt install python3-pip
```

### 2. Install mkdocs
```
pip install mkdocs
pip3 install mkdocs-material
```

## Run

Clone repository

```
git clone {link_repo}
cd {dir_project} 
```

Build & Running Web

```
mkdocs build
mkdocs serve
```
